# Task Manager

A simple task management application built with Vue 3 and TypeScript.

## Features

- Create, read, update, and delete tasks
- Filter tasks by status (all, active, completed)
- Task prioritization
- Due date tracking
- Task statistics

## Technologies Used

- Vue 3 with Composition API
- TypeScript
- Vite
- Pinia for state management
- Vue Router
- Tailwind CSS
